## Getting Started:
#### Dependencies
- Application uses junit, mockito for testing and lomobok
- All dependencies will be handled by ```build.gradle``` file

#### Build
- Run ```gradle build``` to build the project

#### Run
- Run ```gradle bootrun```
- Above command start the application on localhost:8080
